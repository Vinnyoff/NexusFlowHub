# **App Name**: FashionFlow

## Core Features:

- User Authentication: Secure user authentication system with roles: Admin and Cashier/Employee, managed via Firebase Authentication.
- Product Registration: Product registration with name, brand, model, sizes, price, and quantity. Generates a unique UUID token and barcode for each size variant.
- Automated Barcode Generation: Automatically generates Code 128 or EAN-13 barcodes based on the unique product token or ID.
- Point of Sale (POS): A simplified POS interface to scan the barcode and register sells using code reading.
- Sales History: A complete sales history including date, sold products, amounts, user (cashier), and quantity.
- AI-Driven Stock Monitoring: An AI tool analyzes sales history and current stock levels to predict when items will run low, automatically generating restock suggestions.
- Comprehensive Reporting: Sales reports by period, top-selling products, total revenue, low stock alerts, and sales by user presented with clear charts and graphs.

## Style Guidelines:

- Primary color: Deep violet (#7957D6) to evoke sophistication and exclusivity.
- Background color: Very light violet (#F2F0F9) to provide a soft and modern backdrop.
- Accent color: Dark violet (#503990) for highlighting key interactive elements and CTAs.
- Headline font: 'Playfair', serif, providing elegance and high-end touch.
- Body font: 'PT Sans', sans-serif, matching with Playfair and proving readability.
- Use clear, modern line icons for navigation and product categorization.
- Minimalist design with a dashboard showcasing key information through cards and charts.
- Subtle transitions and loading animations to enhance user experience.
